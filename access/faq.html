Sobre o projeto

Q. Quais tecnologias foram utilizadas para o desenvolvimento do site?

A: O site foi desenvolvido utilizando as seguintes tecnologias:

  • HTML
  • CSS
  • Bootstrap 4 (pra deixar o site bonitinho)
  • JavaScript (pra adicionar comportamentos ao site)


Q. Qual o objetivo do site?

A: Além de passar na cadeira, o objetivo é que o site pareça/seja um mockup de um guia turístico.
Ao clicar em cada card se abre um modal, responsável por mostrar informações sobre aquele ponto de maneira resumida. Caso o usuário deseje obter mais informações, dentro do próprio modal existe uma seção com mais referências.
Além disso, o usuário pode ver se data na qual ele vai viajar pra Recife é feriado ou não e acompanhar notícias relevantes sobre a cidade.
